Hello Github Pages

This is my hello world post…

I used to have a static generated website with Hugo. Now I wanted to start fresh and take advantage of Github Pages.

But: Github Pages utilizes Jekyll as static site generator. Although I played a little bit with Jekyll, I never grew fond of it and its templating engine. Since I wanted to play with Github Actions as well, I see this site as a POC.

I used following tutorials for my approach:

  • https://gohugo.io/hosting-and-deployment/hosting-on-github/
  • https://ruddra.com/hugo-deploy-static-page-using-github-actions/
